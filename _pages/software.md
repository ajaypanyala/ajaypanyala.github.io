- [TAMM](https://github.com/NWChemEx/TAMM): A distributed high-performance tensor algebra library.
- [ExaChem](https://github.com/ExaChem/exachem): An open-source exascale-ready suite of scalable electronic structure methods.
- [NWChem](https://github.com/nwchemgit/nwchem): An open-source high-performance ab initio computational chemistry package.
- [GlobalArrays](https://github.com/GlobalArrays/ga): A partitioned global address space (PGAS) library providing a shared memory style programming environment for distributed arrays.
- [NWChemEx](https://github.com/NWChemEx): A high-performance, modular, quantum chemistry software package.
